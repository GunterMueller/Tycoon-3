Summary TreeControl not supported

The TreeControl applet provides a tree menu system containing http links to other pages. Each item has a title and can contain an icon. Folder items can have two icons one each for open and close states. Each item can have a http link to another page (the link includes the frame name "_top" as default). And each item can have a status bar message that appears when the cursor is over the item. The full source is online for those that are interested.
